# Tecboard

Landing page responsiva desenvolvida com **HTML5** e **CSS3** como evolução de um projeto realizado durante o curso de Front-End da Alura.

Após concluir a versão proposta nas aulas, realizei uma evolução aplicando boas práticas de desenvolvimento, organização de código e melhorias na estrutura do projeto, com o objetivo de consolidar os conhecimentos adquiridos e aproximar a implementação de padrões utilizados no mercado.

## ✨ Melhorias implementadas

Em relação à versão original, foram aplicadas as seguintes melhorias:

- Estrutura HTML mais semântica;
- Organização das classes utilizando a metodologia **BEM**;
- Reestruturação e organização do CSS por seções;
- Organização dos assets em diretórios específicos;
- Utilização de fontes locais através do `@font-face`;
- Melhor organização do layout e dos componentes;
- Aprimoramento da responsividade;
- Padronização do código para facilitar futuras manutenções.
